# 🔧 UstaGo (Семей-Мастер) - Telegram Бот

Маркетплейс услуг и заказов для города Семей. Платформа соединяет клиентов с проверенными мастерами через удобный Telegram-интерфейс.

## 🌟 Основные функции

### 👤 Для клиентов
- **Поиск мастеров**: Просмотр анкет, отзывов и категорий.
- **Создание заявок**: Возможность описать задачу, указать бюджет и прикрепить фото.
- **Выбор мастера**: Получение откликов и подтверждение исполнителя.
- **Система рейтинга**: Возможность оценить работу мастера и оставить отзыв.

### 🔨 Для мастеров
- **Регистрация профиля**: Анкета с категориями услуг, опытом и портфолио работ.
- **Лента заказов**: Получение уведомлений о новых заявках в выбранных категориях и районах.
- **Управление балансом**: Система баллов для откликов. Пополнение через Crypto (Telegram Wallet) или банковские карты.
- **Гибкие настройки**: Режим «Не беспокоить», выбор районов работы и видимости профиля.

### 👨‍✈️ Панель администратора
- **Динамические настройки**: Управление методами оплаты (Crypto/Bank), реквизитами и текстами прямо из бота.
- **Управление пользователями**: Поиск по ID/Telegram ID, просмотр статистики и баланса.
- **Система блокировок**: Возможность забанить пользователя с автоматическим уведомлением о снятии ограничений.
- **Рассылка (Broadcast)**: Массовая отправка сообщений (Текст + Фото) всем пользователям.
- **Обратная связь (Live Chat)**: Система тикетов с возможностью переписки в реальном времени.

## 🛠 Технологический стек
- **Python 3.10+** (через менеджер пакетов [uv](https://github.com/astral-sh/uv))
- **Aiogram 3.x** - асинхронный фреймворк для Telegram.
- **SQLAlchemy 2.0** + **PostgreSQL** - хранение данных.
- **Alembic** - миграции базы данных.

## 🚀 Быстрый старт

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/lordamotar/UstaGo-bot.git
   cd UstaGo
   ```

2. **Настройте окружение**:
   Создайте файл `.env`. Рекомендуется использовать шаблон из `.env.example`.
   Минимальные настройки:
   ```env
   BOT_TOKEN=your_token_here
   DATABASE_URL=postgresql+asyncpg://user:pass@localhost:5432/db_name
   ADMIN_IDS=1234567,890123
   ```

3. **Запустите проект** (с использованием `uv`):
   ```bash
   uv sync
   uv run python reset_db.py  # Сброс и инициализация БД (если нужно)
   uv run python main.py      # Запуск бота
   ```
