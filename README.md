# 🔧 UstaGo (Семей-Мастер) - Telegram Бот

Маркетплейс услуг и заказов для города Семей. Платформа соединяет клиентов с проверенными мастерами через удобный Telegram-интерфейс.

## 🌟 Основные функции

### 👤 Для клиентов
- **Поиск мастеров**: Просмотр анкет, отзывов и категорий.
- **Создание заявок**: Возможность описать задачу, указать бюджет и прикрепить фото.
- **Выбор мастера**: Получение откликов и подтверждение исполнителя.

### 🔨 Для мастеров
- **Регистрация профиля**: Анкета с категориями услуг, опытом и портфолио работ.
- **Лента заказов**: Получение уведомлений о новых заявках в выбранных категориях.
- **Отклики**: Возможность предложить свои услуги клиентам напрямую.

### 👨‍✈️ Панель администратора
- **Управление пользователями**: Поиск по ID/Telegram ID, просмотр статистики.
- **Система блокировок**: Возможность забанить пользователя (на 24ч, 7дн, месяц, год или навсегда) с автоматическим уведомлением о снятии ограничений по истечении срока.
- **Рассылка (Broadcast)**: Массовая отправка сообщений (Текст + Фото) всем пользователям с режимом предпросмотра.
- **Обратная связь (Live Chat)**: Анонимная система тикетов с возможностью переписки с пользователем в реальном времени.

## 🛠 Технологический стек
- **Python 3.10+** (через менеджера пакетов `uv`)
- **Aiogram 3.x** - современный асинхронный фреймворк для ботов.
- **SQLAlchemy 2.0** - ORM для работы с БД.
- **PostgreSQL** - надежная реляционная база данных.
- **Alembic** - миграции базы данных.

## 🚀 Быстрый старт

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/lordamotar/UstaGo-bot.git
   cd UstaGo
   ```

2. **Настройте окружение**:
   Создайте файл `.env` на основе `.env.example`:
   ```bash
   BOT_TOKEN=8763135642:xxx...
   DB_URL=postgresql+asyncpg://user:pass@localhost:5432/ustago
   ADMIN_IDS=312082048,12345678
   ```

3. **Установите зависимости**:
   ```bash
   uv sync
   ```

4. **Запустите бота**:
   ```bash
   uv run python main.py
   ```

## 🏗 Структура проекта
- `bot/handlers/` - логика обработки сообщений (admin, client, master, support).
- `bot/keyboards/` - сборка кнопок (inline & reply).
- `bot/middlewares/` - посредники (логирование, проверка банов, поддержка чата).
- `database/models.py` - описание таблиц БД.
- `main.py` - точка входа и запуск фоновых задач (разбан-воркер).

## 📊 Миграции
Для применения изменений в структуре базы данных используйте Alembic или встроенные скрипты инициализации.
